= ネットワーク設定はどこで入手できますか。
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
インターネットプロトコル（IP）アドレス、サブネットワークマスク（サブネットマスク）、およびゲートウェイの情報は、ネットワーク管理者から入手できます。

この情報は、コントローラでポートを設定する際に必要となります。
